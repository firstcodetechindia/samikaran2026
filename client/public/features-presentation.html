The Global Olympiad Operating System is a comprehensive, AI-powered online examination platform designed for worldwide olympiad exams. It provides secure, proctored online examinations with real-time analytics and multi-tenant capabilities.

6AI Proctoring System

Real-Time Monitoring

Continuous camera-based monitoring with 2-second intervals:

  • Face Detection: Verifies student presence
  • Multiple Face Alert: Detects additional persons
  • No Face Warning: Alerts when student leaves view
  • Eye Gaze Tracking: Monitors screen focus

Trust Score System

AI calculates trust score (0-100%) based on:

  • Face visibility percentage
  • Eye contact duration
  • Number of violations
  • Tab switch attempts
  • Browser focus events
  • Overall behavior pattern

Voice Warning System

  • Voice Types: Young (18-22), Adult (25-35), Mature Female (40+)
  • Pitch Control: Adjustable for natural-sounding warnings
  • Auto Translation: English to regional languages

Browser Lockdown

Full-screen mode, right-click disabled, copy-paste blocked, tab switching detection.
