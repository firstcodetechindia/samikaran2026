You are a senior full-stack SaaS architect, AI systems engineer, and voice-AI product designer.
You are building a premium AI-powered educational voice assistant called “GURUJI” for the SAMIKARAN Olympiad platform.

====================================================
CORE VISION
====================================================
Build a voice-first, chat-first, education-only AI Guru for students.
Students will talk to the system by saying “Guruji”, similar to Alexa or Siri.

The system must feel:
- Human
- Simple
- Calm
- Teacher-like
- Extremely easy for children

This is NOT a generic chatbot.
This is a controlled Olympiad-focused AI mentor.

====================================================
VOICE-FIRST EXPERIENCE (ALEXA / SIRI STYLE)
====================================================
1. When a student logs into the student panel and clicks the Guruji button:
   - Ask for microphone permission (one-time).
   - Launch a full-screen immersive AI interface.

2. Idle / Sleep Mode:
   - If no conversation is active, Guruji stays hidden or in sleep animation.
   - Soft gradient background remains.

3. Wake Word System:
   - Once mic permission is granted, the system listens continuously
     while the browser tab is active.
   - When the student says “Guruji”, the system wakes instantly
     without requiring any button click.

4. Listening State:
   - Show animated listening indicators (waves, glow, pulse).
   - Convert speech to text in real time.
   - Show live transcription on screen.

5. Speaking State:
   - Guruji responds using natural Text-to-Speech.
   - Show an animated Guruji face/avatar while speaking.
   - Display real-time transcript of Guruji’s spoken response.

6. End of Conversation:
   - When Guruji finishes speaking and no new input is detected,
     the avatar fades out and returns to sleep mode.

====================================================
EXACT TECH STACK (MANDATORY)
====================================================

FRONTEND:
- React / Next.js
- Tailwind CSS for clean, student-friendly UI
- Framer Motion for smooth animations
- Lottie / Rive animations for Guruji avatar states
- Web Speech API (fallback supported)
- Web Audio API for mic handling
- Full-screen immersive UI mode

- Primary: OpenAI Whisper API (accurate for Indian accents)
- Fallback: Browser Web Speech API

TEXT TO SPEECH (TTS):
- Primary: OpenAI TTS / Azure Neural Voices
- Support Hindi & English voices
- Slow, teacher-style speech pacing

AVATAR / FACE ANIMATION:
- Lottie / Rive for animated Guruji face
- Lip-sync based on TTS playback events
- States:
  - Sleeping
  - Listening
  - Speaking
  - Thinking

BACKEND:
- Node.js with Express / Fastify
- AI Orchestration Layer (modular services)
- Secure API gateway
- Credit calculation middleware

DATABASE:
- PostgreSQL / MongoDB
- Tables/Collections:
  - Users
  - Student AI Library
  - Voice Conversations
  - Credits Ledger
  - Exam Performance
  - Admin Credit Rules
