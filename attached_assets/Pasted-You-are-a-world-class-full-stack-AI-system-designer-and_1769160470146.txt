You are a world-class full-stack AI system designer and visual UX architect. 

Your task is to build a best-in-the-world **Chatbot Flow Builder** inside Replit — a **no-code, drag-and-drop, highly intuitive visual tool** that allows anyone — even users with zero technical experience — to design complex chatbot conversations easily.

This Flow Builder must support **every possible chatbot requirement**, including conditions, loops, variables, triggers, fallback logic, multilingual text, scores, state, API integrations, timers, audio triggers, and advanced routing.

## 🧠 1. VISUAL FLOW BUILDER UI

### Core Features
• **Drag-and-drop nodes** (Start, Message, Input, Condition, Loop, API, NLP Intent, Fallback, Redirect, End)  
• **Connectors** with multiple branches  
• Unlimited branching and nested flows  
• **Color coded nodes** (Info, Logic, API, NLP, Decision)  
• **Zoom + pan + multi-select + grid snap**  
• **Real-time simulation/testing sandbox**  
• **Undo/Redo + version history**  
• **Search within flow**  

### Node Types (with rich settings)
• **Start Node** – Entry point  
• **Message Node** – Static or dynamic text  
• **User Input Node** – Free text, choice, numeric, regex  
• **Condition Node** – Branch based on variables or NLP intent  
• **Loop Node** – Repeat until condition  
• **Delay/Timer Node** – Wait or remind later  
• **API Node** – Call external APIs, save responses  
• **NLP Intent Node** – Use AI intent classification  
• **Fallback Node** – Graceful fallback with retry paths  
• **Redirect Node** – To other flows or sub-flows  
• **Exit Node** – End of conversation  

### UX Specifications
• Node palette sidebar with drag icons  
• Canvas center workspace  
• Property inspector panel on selection  
• Inline editable titles/text  
• Click-to-test individual node paths  
• Flow export/import (JSON)  
• **Preview chat panel** for live test

Each node supports **conditions, variables, language variants, and rich text**. The builder must manage context state and allow previewing the full conversation. :contentReference[oaicite:1]{index=1}
