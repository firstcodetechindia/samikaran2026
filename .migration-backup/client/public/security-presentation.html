5 Anti-Cheating Measures

R Question Randomization

Each student receives questions in unique random order using Fisher-Yates shuffle.

O Option Shuffling

MCQ options randomly ordered. "Option A" for one student may be "Option C" for another.

T Strict Time Limits

Enforced exam duration with automatic submission when time expires.

1 Single Attempt

Students get only one attempt per exam. No restart or retake options.

How Randomization Works

The Fisher-Yates shuffle algorithm ensures:

  • Each student sees questions in a different sequence
  • MCQ options are independently shuffled per student
  • Correct answer positions are completely randomized
  • Screen sharing becomes ineffective as sequences differ

All warnings are written in English and automatically translated to regional languages using OpenAI GPT-4o. Admins can also manually edit translations.
