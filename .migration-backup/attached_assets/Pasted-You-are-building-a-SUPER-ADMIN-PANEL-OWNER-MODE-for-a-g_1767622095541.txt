You are building a SUPER ADMIN PANEL (OWNER MODE) for a global, AI-powered, multi-tenant Olympiad Examination SaaS platform.

This panel is accessible ONLY to the platform owner (root level) and provides full global control over users, exams, analytics, AI, finance, security, SaaS tenants, and infrastructure.

The system must support real-time analytics using WebSockets (Socket.IO).

--------------------------------------------------

--------------------------------------------------
- Global platform ON/OFF (maintenance mode)
- Region-wise enable/disable (Country, State, Board, District)
- Language enable/disable per region
- Exam season control (Open / Lock / Archive)
- Emergency lockdown (security, cheating, legal, government notice)
- Global configuration overrides (time limits, rules, scoring)
- Feature flags (enable/disable features without redeploy)
