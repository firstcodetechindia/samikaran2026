I want to implement a SMART, COST-FREE, RULE-BASED CHATBOT SYSTEM in my platform with ZERO external AI cost (NO OpenAI, NO paid AI APIs).

GOAL:
Chatbot intelligent lage, user ko samjhe, guide kare aur help kare — bina kisi AI model ke.
Pure system-driven, logic-based, fast aur safe hona chahiye.

================================================
CHATBOT MODES
================================================

Chatbot must work in TWO MODES:

1️⃣ GUEST MODE (User not logged-in)
2️⃣ LOGGED-IN MODE (User authenticated)

================================================
MODE 1: GUEST CHATBOT (NOT LOGGED-IN USERS)
================================================

AVAILABLE CONTEXT:
- User is NOT logged-in
- Current page (Home, Olympiad Info, Pricing, etc.)
- Device type (mobile / desktop)

BEHAVIOR:
- Friendly, welcoming, generic greeting
- No personal data usage
- No assumptions

Greeting Examples:
- "Hi 👋 Main aapki Olympiad journey me madad kar sakta hoon."
- "Welcome! Olympiad exams ke baare me poochna chahoge?"

ALLOWED HELP:
- Olympiad information
- Exam pattern & structure
- Eligibility & registration process
- Pricing & general FAQs
- How online exam works

RESPONSE LOGIC:
- Predefined intent-based responses
- Keyword + intent matching
- Button-based quick replies

SOFT LOGIN NUDGE:
- Every few interactions suggest login politely
Example:
"Personalised guidance ke liye login karein 👉 Login / Sign up"

RESTRICTIONS:
- No results
- No performance
- No user-specific answers

================================================
MODE 2: LOGGED-IN CHATBOT (SMART MODE)
================================================

USER CONTEXT FETCH (ON CHAT OPEN):
- User first name
- User role (Student / Parent / Teacher / School / Partner)
- Role-specific allowed data
- Recent activity (last exam, last report, pending action)

This context must be stored in session memory.

SMART GREETING:
- Use user's first name
- Role-aware greeting

Examples:
- "Hi Rahul 👋 Aaj kaise improve karna chahoge?"
- "Hello Mrs. Sharma, aapke child ka latest progress ready hai."

Use RULE-BASED INTELLIGENCE:

- Keyword detection

NO AI TEXT GENERATION.
